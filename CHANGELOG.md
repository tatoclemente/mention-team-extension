# Changelog

All notable changes to the "Mention my team" extension will be documented in this file.

## [0.1.0] - 2025-11-22

### Added
- Initial beta release
- Mention team members with `@username` syntax in code comments
- Automatic GitHub Issue creation with assignment
- Email notifications via GitHub
- Direct links to code with line numbers
- Code context (10 lines) included in issues
- Support for `//? author: @username fecha: DD MMM YYYY` metadata
- Configuration commands for GitHub token and repository
- Label `team-mention` for easy filtering
- Support for multiple programming languages (JS, TS, Python, Java, Go, etc.)

### Notes
- This is a beta release. Please report issues on [GitHub](https://github.com/tatoclemente/mention-team-extension/issues)
